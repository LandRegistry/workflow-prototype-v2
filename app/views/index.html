Information for designers

How to use this repository and prototyping kit ...
  • Workflow layout templates

    A list of base templates already coded, available to use. Copy these and add to your own folder (directory) in the 'views' directory:

    • index-main (opens in new tab).
      A standard page template using the layout template of main.html
    • index-wf (opens in new tab).
      A standard page template using the primary Workflow layout template of wf_layout.html
    • index-wf-legacy (opens in new tab).
      A page template using the legacy Workflow layout template of index-wf-legacy.html. Used for recreating pages to match older Workflow pages. (The orginal legacy prototype used different CSS classes and IDs on the footer).
    • Additional layouts in the 'layouts' folder (directory) include wf_layout_legacy_wcag_updates.html and wf_layout_legacy_wcag_updates-new-autocomplete.html, which include specific CSS and JS for key functionality.
  • Clearing session data (Tip)

    A solution for more convenient session data clearing is to use the 'Clear data link' plug-in. Simply add the following code to the bottom of your layout page:
    <script type="text/javascript" src="/public/javascripts/clear-data-link.js"></script>

    This appends a link to the list of links in the footer of your pages. Clicking the new 'Clear data (popup)' link opens a popup window: Select 'Clear the data', close the pop-up window and then refresh the page.
